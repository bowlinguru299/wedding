Installing Template.

Carna is HTML template so there's no really "installation" required just unzip the main file you've downloaded from themeforest and after unziping you have your all template ready to be used.

HTML Structure

This Template has a Responsive layout and is based on the Bootstrap 3 Framework. For more information about this visit Bootstrap Scaffolding.

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Given this example, we have .col-md-8 and ..col-md-4, making for 12 total columns and a complete row.

EXTRA CARNA

In carna its very easy to initialize multible master slider , you need only master-slider class in your html page and to define the feature you need throgh data attribute in that class

  1. slider options
    Name Default Value Description
    data-width="1170" Define the slider width
    data-height="650" Define the slider height
    data-layout="fillwidth" Specifies layout of the slider, it accepts following values:
    • boxed: The default layout of slider, size of the slider does not exceed the specified width and height in slider options.
    • fullwidth: Forces the slider to adapt width to the browser width.
    • fullscreen: Forces the slider to adapt width and height to the browser window dimension.
    • fillwidth: Enables the slider to adapt width to its parent element.
    • autofill: Enables the slider to adapt width and height to its parent element.
    • partialview: It's like the boxed layout but nearby slides are always visible.
    data-autoplay false true or false
    data-view basic fade, mask, wave, flow, stock, scale, focus, parallaxMask, partialWave, fadeBasic, fadeWave, fadeFlow
    data-space 0 The spacing value between slides in pixels.
    data-loop false Enables the continuous sliding mode / false or true
    data-wheel false Enables mouse scroll wheel navigation.
    data-keyboard false Enables keyboard navigation.
    data-full-screen-margin 0 Specifies margin space at the bottom of slider, it only affects with "fullscreen" layout.
    data-full-auto-height false Whether the slider adapts its height to each slide height or not. It overrides heightLimit option.
    data-speed 17 Specifies slide changing speed. It accepts float values between 0 and 100.
    data-shuffle false Enables the shuffle slide order / false or true
    data-dir "h" Specifies slide changing direction. It accepts two values "h" (horizontal) and "v" (vertical).
    data-over-pause false Whether the slider pauses slideshow on hover..
    data-instant-start-layers false Whether start show transition of layers before slide transition complete.
    data-filter false Applys CSS3 filters to slide transition views. For more information please check this document.
    data-control-scrollbar false Whether start show transition of layers before slide transition complete.
  2. slider controll
    • Arrows :
      Name Default Value Description
      data-control-arrows false Adds arrow navigation to the slider
      data-arrows-hide" null Hides the control if width of slider comes less than specified.
      data-arrows-autohide" true Whether the control hides when mouse leaves the slider.
      data-insert-arrow-to" insert the class name of the slider , this is will push the arrows to the edge
    • bullets :
      Name Default Value Description
      data-control-bullets false Adds bullets navigation to the slider
      data-bullets-dir" h Specifies the direction of control. It accepts two values "h" (horizontal) and "v" (vertical).
      data-bullets-align" bottom Aligns the control to in or out of slider. It accepts "top" and "bottom" values.
    • thumblist :
      Name Default Value Description
      data-control-thumblist false Adds thumblist to the slider
      data-thumbs-autohide false Whether the thumb hides when mouse leaves the slider.
      data-thumbs-dir h Specifies the direction of control. It accepts two values "h" (horizontal) and "v" (vertical)
      data-thumbs-type thumbs Specifies the type of control, it can used as "thumbs" or "tabs"
      data-thumbs-width 100 Specifies the width of each thumb in pixel.
      data-thumbs-height 100 Specifies the height of each thumb in pixel.
      data-thumbs-align bottom Aligns the control to in or out of slider. It accepts "top", "bottom", "left" and "right" values.
      data-thumbs-space 10 Specifies the space between thumbs or tabs.
      data-thumbs-margin 10 Specifies the space between the control and slider in pixel.
      data-thumbs-inset false Whether insert the control inside or outside of slider.
      data-thumbs-hideunder 600 Hides the control if width of slider comes less than specified.

Carousel Opations:

To set bx slider into carousel mood you need to set data-bx-carousel="true" here is the opations you need:

Name Default Value Description
data-bx-carousel false boolean (true / false)
data-max-slide 1 The maximum number of slides to be shown. Slides will be sized up if carousel becomes larger than the original size.)
data-slide-width 0 The width of each slide

The width of each slide will be measured automatically based on the max slide number you insert , this work when the viewport is over 1200px , so nevertheless you need to insert the width to be effective when the viewport is beneath 1200px

data-move-slide 0 The number of slides to move on transition If zero (default), the number of fully-visible slides will be used.
